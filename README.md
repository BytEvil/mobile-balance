Mobile Balance
==============

Installation
------------

Do as usual: `pip install mobile-balance`

Usage
-----

Specify mobile operator and credential from it's personal profile.
For example:

    $ mobile-balance mts --phone=9154041111 --password=to-secret-to-share
    196.513792
